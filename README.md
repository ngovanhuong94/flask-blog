### flask-blog
A simple blog using Flask framework
### Prerequisites

* python
* virtualenv


1. Clone this repository

2. Install dependencies
    ```bash
    $ cd flask-blog
    $ virtualenv venv
    $ pip install requirements.txt
    ```

    ```

## Run the app

1. Start the server
    ```bash
    $ flask run
    ```
2. Browse to `http://localhost:5000/`
